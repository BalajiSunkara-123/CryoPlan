# CryoPlan Services Directory

This folder is reserved for external sensor feeds and database synchronization connectors:
- `ohrcTelemetryService.ts` - streaming high-resolution imagery blocks from Chandrayaan orbiters.
- `lolaElevationService.ts` - fetching laser altimeter digital terrain profiles from ground servers.
- `missionDatabaseService.ts` - syncing waypoint vectors with SAC core databases.
